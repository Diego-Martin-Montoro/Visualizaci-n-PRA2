1.3 Reestructuración del dataset¶

Como podemos observar, la variable vaccines aparece como una lista de nombres de vacunas separadas con el carcter ,. Vamos a generar una fila por cada nombre de vacuna (para cada registro), replicando el resto de los datos del registro tantas veces como vacunas haya. Para el caso que nos ocupa, los nicos datos que nos interesan son: el pas, la vacuna y la fecha del primer registro de dicho pas y dicha vacuna (de manera que podemos saber cundo un pas comenz a usar una vacuna determinada).

Para ello, comenzamos agrupando por pas y quedndonos siempre con el primer registro del grupo. Como estos estn ordenados por fecha, nos estaremos quedando con el primer registro de cada pas. Acto seguido, proyectamos los campos de nuestro inters: vaccines, country y date.

pais_vacunas_string = vacunas_df.groupby('country').first( )

pais_vacunas_string = pais_vacunas_string[['country','vaccines','date']]

Mediante este bucle, iteramos los nombres de las vacunas existentes en el campo vaccines para cada pas para replicar la fecha y el nombre de este tantas veces como vacunas tenga. NOTA: en el conjunto de datos dado un pas su lista de vacunas usadas no vara.
